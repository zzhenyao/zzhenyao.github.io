与c++底层的绑定方法

PYBIND11_MODULE 宏绑定函数

该宏用于定义一个 PyBind11 模块，example 是python模块的名称，m 是模块对象，可以用于绑定类和函数，以便在 Python 中使用。

py::class_ 绑定c++类

pybind11::class_<MyClass>(m, "MyClass")
“MyClass”: 是py中的新类名

: 指的绑定c++的类名

- m.def 是用于将c++全局函数绑定到 Python 模块上的方法。
-    .def 是用于将类的方法绑定到 Python 上的接口。
